SMF Benutzer Hilfe: Registrieren


Übersicht • Registrieren • Ein-/Ausloggen • Profil • Schreiben • Private Mitteilungen • Suchen

Wenn Sie Ihr SMF Forum installiert haben oder einem anderen SMF Forum beitreten möchten, sollten Sie einen Benutzernamen registrieren. Dies ermöglicht es Ihnen, eine gleichbleibende Identität für Ihre Beiträge zu nutzen und zu sehen, was Sie schon gelesen haben, egal von wo Sie arbeiten und welchen Computer Sie benutzen. Sie werden in der Lage sein, ein persönliches Profil zu erstellen und das eingebaute System für Private Mitteilungen zu benutzen. Ausserdem können Sie sich Benachrichtigungen schicken lassen, Ihre Aktivitäten im Forum kontrollieren und vieles mehr.

Wie Sie sich registrieren können

Wenn Sie als Gast in einem Forum sind, sollten Sie eine dieser beiden Dinge sehen können:

  • Irgendeine Seite des Forums, in diesem Fall sollten Sie den Registrieren Button aus dem Hauptmenü.
  • Ein Login-Bildschirm mit einer Warnung darüber, dass nur registrierte Mitglieder das Forum nutzen können. In diesem Fall sollten Sie entweder den dort befindlichen Registrations-Link benutzen, oder den Registrieren Button aus dem Hauptmenü.
Warnung!
Nur registrierte Mitglieder haben Zugriff auf diesen Bereich.
Bitte einloggen oder registrieren Sie einen neuen Zugang mit Forum Name.

Der Registrierungs Bildschirm

Registrierung - Notwendige Angaben
Wähle Benutzername::
Wird nur zur Identifizierung von SMF verwendet. Spezialzeichen können Sie nach dem Login verwenden, indem Sie den Benutzernamen im Profil ändern.
E-Mail:
Es muss eine gültige E-Mail Adresse sein.
Passwort wählen::
Passwort wiederholen::
Hier werden die Nutzungsbedingungen angezeigt, sofern welche vorhanden sind!


Wenn Sie den Registrierungs Bildschirm sehen, müssen Sie einen Benutzernamen, eine E-Mail Adresse und ein Passwort eingeben und könnten die Möglichkeit haben, Ihre E-Mail Adresse nicht anzeigen zu lassen. Da Sie in den meisten Foren einer Nutzungsbedingung zustimmen müsssen um sich registrieren zu können, lesen Sie die Bedingungen und klicken dann Ich bin einverstanden bevor Sie den Registrieren Button betätigen. Was nun passiert hängt, von den Einstellungen des Forums ab. Falls Sie einen Login Bildschirm sehen sind Sie registriert und können sich nun Einloggen. Ist dies nicht der Fall, bestehen zwei weitere Möglichkeiten:

  • Wenn die Registrierung so eingestellt ist, dass eine E-Mail Aktivierung nötig ist, dann sollte Ihnen dies nun auf dem Bildschirm mitgeteilt werden und Sie sollten eine E-Mail mit weiteren Anweisungen erhalten haben.
  • Wenn die Registrierung so eingestellt ist, dass eine Genehmigung von einem Forum Administrator nötig ist, dann sollte Ihnen dies nun auf dem Bildschirm mitgeteilt werden und Sie sollten eine Mitteilung per E-Mail erhalten haben. Später werden Sie eine E-Mail erhalten, in der Ihnen mitgeteilt wird, ob Ihre Registrierung genehmigt wurde oder nicht.
